# Teslo API

1. Clonar proyecto
2. ```npm i```
3. Clonar el archivo ```.env.template``` y renombrarlo  ```.env```
4. Cambiar Variables de Entorno
5. Levanta la base de datos
```
docker-compose up -d
```
6. Ejecutar SEED

```
http://localhost:3000/api/seed
```
7. Levantar: ```npm run start:dev```
